# Sales AI Dashboard

**Goal:** Build an AI-powered sales & inventory analytics dashboard that supports natural-language questions, forecasting, and interactive visualizations.

## Project structure
- `data/` - raw and cleaned datasets (not tracked in git)
- `notebooks/` - EDA and experimentation notebooks
- `src/` - scripts and app code
- `charts/` - exported visuals
- `reports/` - generated PDF reports

## Quick start (local)
1. Create and activate venv:
   - `python -m venv .venv` (then activate)
2. Install:
   - `pip install -r requirements.txt`
3. Run notebook:
   - `jupyter lab` or `streamlit run src/app.py`
